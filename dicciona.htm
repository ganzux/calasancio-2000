accesibilidad

Característica de un sistema que incorpora hardware o software para facilitar su uso por parte de personas con alguna incapacidad, como movilidad limitada, ceguera o sordera.

Active Data Objects

(ADO) Componentes que permiten a las aplicaciones cliente tener acceso y manipular datos de una base de datos de un archivo o un servidor a través de un proveedor.

ActiveX

Conjunto de tecnologías que permite la interacción de los componentes del software en un entorno de red, con independencia del lenguaje en el que se hayan creado. ActiveX se utiliza principalmente para desarrollar contenido interactivo para el World Wide Web, aunque puede usarse en aplicaciones de escritorio y otros programas. Consulte también controles ActiveX.

archie

Servicio de búsqueda de archivos (de texto, gráficos, música...) que suelen transferirse vía FTP.

 
archivo

Colección con nombre de información que se almacena en un equipo.

artículo

Mensaje o entrada de un grupo de discusión o de noticias de Internet. Un artículo puede ser una respuesta a un artículo anterior.

ASCII

(American Standard Code for Information Interchange, Código americano estándar para el intercambio de información) Esquema de codificación que utiliza 7 u 8 bits para asignar valores numéricos a un máximo de 256 caracteres, incluyendo letras, números, signos de puntuación, caracteres de control y otros símbolos. El esquema ASCII se desarrolló en 1968 para estandarizar la transmisión de datos entre los diferentes sistemas de hardware y software, y está presente en todos los equipos personales.

ASP

(Página Active Server) Documento que contiene secuencias de comandos incrustadas del servidor. Los servidores Web compatibles con ASP pueden ejecutar estas secuencias de comandos. Para el cliente, una ASP es un documento HTML estándar que puede verse en cualquier plataforma mediante cualquier explorador de Web.

En un entorno multiusuario o de red, proceso por el cual el sistema valida la información de inicio de sesión de un usuario. Se comparan el nombre y la contraseña del usuario con respecto a una lista de autorización y, si el sistema confirma la coincidencia, se concede el acceso en el grado especificado en la lista de permisos para dicho usuario.

Autenticación de contraseña distribuida (DPA)

Método avanzado y distribuido de autenticación de usuarios de Microsoft que permite un único inicio de sesión. Microsoft Membership System proporciona la compatibilidad con DPA, optimizado para las necesidades de los proveedores de servicios Internet y los servicios en línea.

base de datos de autenticación

Base de datos de un servidor que comprueba la coincidencia de los nombres de usuario con las contraseñas.

base de datos relacional

Un sistema de administración de bases de datos relacionales (RDBMS) organiza los datos en filas y columnas relacionados según especifica un modelo relacional. Microsoft SQL Server y Oracle son ejemplos de sistemas de administración de bases de datos relacionales. Una base de datos de Microsoft Access es un ejemplo de base de datos relacional.

beta

Es una versión no definitiva de un programa. Estas versiones se suelen distribuir de forma gratuita a todos los usuarios que lo soliciten (o a usuarios registrados y programadores), para de ésta forma localizar de manera más rápida los posibles errores que pueda contener. La primera versión es la llamada Beta 1 y así correlativamente hasta llegar a la versión definitiva  de la aplicación.

 

biselado

BMP

(mapa de bits) Formato estándar de archivos gráficos en equipos compatibles con Windows. Los gráficos de mapa de bits se pueden guardar para sistemas Windows u OS/2 y admiten color de 24 bits. 

botón de comando

Campo de formulario en el que un visitante del sitio puede hacer clic para enviar un formulario o restablecer un formulario a su estado inicial.

botón de opción

Campo de formulario que presenta una selección que se puede elegir si se hace clic en un botón. Los botones de opción se presentan en una lista y uno de ellos está seleccionado de forma predeterminada. Si elige un nuevo miembro de la lista, se anulará la selección del elemento actual. Consulte también casilla de verificación.

bug

Un "bug" (que significa "bicho" en inglés) es un pequeño error que escapó a los creadores de un programa o aplicación y que es detectado después. Puede no ser muy importante y no influir en el funcionamiento de éste, ser más peligroso e impedir el correcto funcionamiento o afectar a la seguridad de su sistema.

campo de formulario

Campo de entrada de datos en una página. Un visitante del sitio proporciona información en un campo al escribir texto o seleccionar un campo.

campo de menú desplegable

Elemento de campo de formulario que presenta una lista de opciones en un menú desplegable. Se puede configurar un campo de formulario de menú desplegable para que permita la selección de varios campos o de un único campo.

campo oculto

Campo de formulario que es invisible para un visitante del sitio pero que proporciona datos a un controlador de formularios. Cada campo oculto se implementa como un par nombre-valor. Cuando un visitante del sitio envía un formulario, se pasan sus campos ocultos al controlador de formularios junto con los pares nombre-valor de cada campo visible del formulario. Consulte también par nombre-valor.

carpeta

Área de almacenamiento con nombre de un equipo que contiene archivos y otras carpetas.

casilla de verificación

Campo de formulario que se puede activar si se hace clic en una casilla. Cuando una casilla está activada, normalmente se muestra con una marca de verificación o X. Las casillas de verificación suelen estar agrupadas para representar un conjunto de opciones no exclusivas. Consulte también botón de opción.

Tabla estándar que asocia un índice numérico a cada carácter de un juego de caracteres. La tabla se utiliza cuando crea una página Web para su uso en un idioma específico. También se denomina página de códigos.

Botón animado de la barra de exploración de una página Web que se activa cuando se mueve el mouse (ratón) sobre el botón o cuando se hace clic en el botón. Consulte también barra de exploración.

En el World Wide Web, cadena de caracteres que permite a un visitante del sitio tener acceso a un servicio Internet, como FTP, si dicho servicio requiere autenticación. correo electrónico

Sistema que permite enviar y recibir mensajes escritos a través de la red de Internet a cualquier otro usuario de éste sistema independientemente de su situación geográfica. Dentro de cada mensaje de correo electrónico es posible incorporar también ficheros de todo tipo, desde imágenes hasta programas. Un ejemplo de dirección de correo electrónico es alvaro@alcedo .net: alvaro es el nombre del usuario, alcedo es el servidor donde se almacenan los mensajes y net es la extensión. Algunos analistas aseguran  que en algunos años el correo electrónico sustituirá al correo postal.

cuadro de texto con desplazamiento

Campo de formulario etiquetado, de varias líneas, en el que los visitantes del sitio pueden escribir una o más líneas de texto.

cuadro de texto de una línea

dirección IP

(Dirección de Protocolo de Internet) Forma estándar de identificar un equipo que está conectado a Internet, de la misma manera que un número de teléfono identifica un teléfono en una red telefónica. La dirección IP consta de cuatro números separados por puntos y cada número es inferior a 256; por ejemplo, 192.200.44.69. Su proveedor de servicios Internet o el administrador del servidor Web asignará a su equipo una dirección IP.

dirección URL

(Localizador de recursos universal) Cadena que proporciona la dirección de Internet de un sitio Web o un recurso del World Wide Web, junto con el protocolo mediante el cual se tiene acceso al sitio o al recurso. El tipo más común de dirección URL es http://, que proporciona la dirección de Internet de una página Web. Otros tipos de dirección URL son gopher://, que proporciona la dirección de Internet de un directorio Gopher, y ftp://, que proporciona la ubicación de red de un recurso FTP.

dirección URL absoluta

Dirección completa de Internet correspondiente a una página o recurso del World Wide Web. La dirección URL absoluta incluye un protocolo, como "http", una ubicación en la red y una ruta de acceso y un nombre de archivo opcionales. Por ejemplo, http://ejemplo.microsoft.com/ es una dirección URL absoluta. Consulte también URL.

dirección URL relativa

Dirección de Internet de una página u otro recurso del World Wide Web relativa a la dirección de Internet de la página actual. Una dirección URL relativa proporciona la ruta de acceso desde la página actual a la página o el recurso de destino. Una dirección URL relativa puede incluir un protocolo. Por ejemplo, la dirección URL relativa doc/ejemplo.htm se refiere a la página ejemplo.htm de la carpeta doc, bajo la carpeta actual. Consulte también URL.

efecto de transición

Uno de varios efectos de presentación de páginas, como Disolver o desvanecimiento en negro, que están disponibles en algunos exploradores de Web. Se pueden configurar los efectos de transición para que ocurran cuando un usuario visite o abandone una página.

encabezado

Estilo de párrafo que se muestra con un tipo de letra mayor que el texto normal. El tamaño de un encabezado está relacionado con su nivel: Encabezado 1 es el de tamaño máximo, Encabezado 2 el de tamaño siguiente, y así sucesivamente. Utilice encabezados para asignar nombres o títulos a párrafos o páginas completas.

EPS

(PostScript encapsulado) Extensión del formato de archivos gráficos PostScript desarrollado por Adobe Systems. EPS permite incorporar archivos gráficos PostScript a otros documentos.

espaciado entre celdas

Cantidad de espacio existente entre las celdas de una tabla. El espaciado entre celdas es el grosor, en píxeles, de las paredes que rodean a cada celda.

estilo

Los estilos se utilizan para controlar la fuente, la alineación y el espaciado del texto, la apariencia de las páginas de fondo y otros atributos HTML. Una colección de estilos se denomina hoja de estilos. Consulte también hoja de estilos en cascada.

estilo en línea

Método para aplicar propiedades y valores de hoja de estilos en cascada a un elemento de una página, como una tabla, un gráfico o un control ActiveX. Puede utilizar este método incluso aunque la página no esté vinculada a una hoja de estilos externa ni contenga una hoja de estilos incrustada.

explorador de Web

Aplicación que interpreta el código de marcado de los archivos HTML, les da formato de página Web y los muestra al usuario. Algunos exploradores también permiten a los usuarios enviar y recibir correo electrónico, leer grupos de noticias y reproducir archivos de sonido o vídeo que se hallan incrustados en los documentos Web.

file

Protocolo de Internet que hace referencia a archivos de un disco o de una red de área local. 

formulario

Conjunto de campos de entrada de datos en una página que se procesan en un servidor Web. Los datos se envían al servidor cuando un visitante del sitio envía el formulario al hacer clic en un botón o, en algunos casos, al hacer clic en un gráfico.

FTP

(Protocolo de transferencia de archivos) Servicio de Internet que transfiere archivos de un equipo a otro a través de líneas telefónicas estándar. 

FTP anónimo

Capacidad de tener acceso a un equipo remoto en el que no se dispone de cuenta, mediante el Protocolo de transferencia de archivos (FTP) de Internet. Con FTP anónimo, los usuarios tienen derechos de acceso restringidos y, en general, sólo pueden ver una lista, examinar o copiar archivos a o desde un directorio público del sistema remoto. La mayoría de los sitios FTP no permiten acceso FTP anónimo con el fin de mantener la seguridad. Consulte también FTP.

GIF

(Formato de intercambio de gráficos) Formato de archivos gráficos que se utiliza comúnmente para mostrar gráficos indizados por color en el World Wide Web. GIF es un formato comprimido diseñado para reducir al mínimo el tiempo de transferencia de archivos a través de líneas telefónicas estándar. 

GIF animado

Archivo que contiene una serie de gráficos GIF (Formato de intercambio de gráficos) que algunos exploradores de Web muestran en una secuencia rápida, proporcionando un efecto de animación. Consulte también GIF.

GIF entrelazado

Imagen en formato GIF que se muestra gradualmente en un explorador de Web, mostrando versiones cada vez más detalladas de la imagen hasta que se termina de descargar todo el archivo. Consulte también GIF.

hipertexto

Originalmente, cualquier información textual de un equipo que contiene saltos a otra información. Los saltos de hipertexto se denominan hipervínculos. En el World Wide Web, el hipertexto es la forma principal de explorar páginas y sitios Web. El hipertexto de las páginas Web se ha ampliado hasta incluir hipervínculos desde texto, imágenes o gráficos y mapas de imágenes.

hipervínculo

Puntero desde un texto, desde una imagen o un gráfico, o desde un mapa de imagen a una página o un archivo del World Wide Web. En el World Wide Web, los hipervínculos son la forma principal de explorar páginas y sitios Web. También se denomina vínculo.

hipervínculo activo

Hipervínculo seleccionado actualmente en un explorador de Web. Algunos exploradores de Web indican el hipervínculo activo con un cambio de color.

hipervínculo externo

Hipervínculo que señala a una página o un archivo que está fuera del Web actual.

hipervínculo predeterminado

En un mapa de imagen, el hipervínculo que se sigue cuando un visitante del sitio hace clic en un área de la imagen donde no hay ninguna zona activa. Consulte también zona activa.

hipervínculo visitado

Hipervínculo de una página que se ha activado. Los exploradores de Web suelen mostrar los hipervínculos visitados en un color especificado.

Hoja de estilos en cascada

(CSS) Especificación HTML desarrollada por el World Wide Web Consortium que permite a los autores de páginas Web adjuntar hojas de estilos a los documentos HTML. Las hojas de estilos pueden incluir información tipográfica acerca de la apariencia de la página, como la fuente del texto de la página. Las CSS también controlan cómo se combinan las hojas de estilos de los documentos HTML con el estilo del usuario.

hoja de estilos externa

Una hoja de estilos en cascada de un archivo con una extensión .css. El archivo .css está compuesto únicamente de reglas de estilo en una sintaxis válida de .css, sin ninguna etiqueta HTML alrededor. Al definir estilos en una o más hojas de estilos externas y vincularlos a páginas de su Web se asegura una apariencia coherente de dichas páginas. Si cambia un estilo en una hoja de estilos externa, dicho cambio se reflejará en todas las páginas vinculadas a esa hoja de estilos.

hoja de estilos incrustada

Hoja de estilos en cascada que está incrustada en una página. Los estilos de una hoja de estilos incrustada sólo pueden aplicarse a la página que contiene la hoja de estilos y extenderán o suplantarán a los estilos definidos en cualquier hoja de estilos externa que esté vinculada a la página.

HTML

(Lenguaje de marcado de hipertexto) Lenguaje estándar de marcado empleado para documentos del World Wide Web. El desarrollo del lenguaje HTML se realiza en el World Wide Web Consortium. El lenguaje HTML indica mediante etiquetas cómo deben mostrar los exploradores de Web los distintos elementos de las páginas, como texto e imágenes, así como la forma en que dichos exploradores deben responder a las acciones del usuario, como la activación de hipervínculos, cuando presiona una tecla o hace clic con el mouse. La mayoría de los exploradores de Web, especialmente Microsoft Internet Explorer y Netscape Navigator, reconocen más etiquetas HTML que las incluidas en el estándar actual.  Consulte también World Wide Web Consortium.

HTML dinámico

(DHTML) Extensión del lenguaje HTML que permite agregar efectos de presentación al texto y a los objetos.

HTTP

(Protocolo de transporte de hipertexto) Protocolo de Internet que permite a los exploradores de Web recuperar información desde servidores del World Wide Web.

imagen

Archivo gráfico que se puede insertar en una página Web y mostrar en un explorador de Web. También se denomina imagen.

inclusión del servidor

Internet

Red mundial formada por equipos, redes y gateways que utilizan protocolos TCP/IP para comunicarse entre sí. En el núcleo de Internet hay líneas de comunicación de datos de alta velocidad entre los principales equipos host, compuestas por miles de sistemas comerciales, gubernamentales, educativos y de otro tipo, que enrutan los datos y los mensajes. En la actualidad, Internet ofrece una gama muy diversa de servicios a los usuarios, como correo electrónico, World Wide Web, FTP, grupos de noticias de usuarios, Gopher, IRC, telnet, etc.

intranet

Red diseñada para el proceso de información dentro de una organización. Entre sus aplicaciones se incluyen servicios como la distribución de documentos y software, el acceso a bases de datos y entrenamiento. El nombre de intranet procede de que generalmente emplea aplicaciones asociadas a Internet, como páginas Web, exploradores de Web, sitios FTP, correo electrónico, grupos de noticias y listas de distribución, a las cuales sólo pueden tener acceso los usuarios de la organización. Consulte también servidor de seguridad.

IP

(Protocolo de Internet) Software de Internet que divide los datos en paquetes para su transmisión a través de Internet. Los equipos deben ejecutar IP para poder comunicarse a través de Internet. Consulte también TCP.

ISAPI

(Interfaz de programación de aplicaciones de servidor de Internet) Interfaz para el desarrollo de aplicaciones de servidor Web desarrollada por Process Software y Microsoft Corporation, que puede utilizarse en lugar de CGI.

Java

Lenguaje de programación de propósito general creado por Sun Microsystems. Actualmente, Java se utiliza principalmente para crear pequeñas aplicaciones, o subprogramas, para el World Wide Web. Consulte también subprograma Java.

JavaScript

Lenguaje de secuencias de comandos desarrollado por Netscape Communications y Sun Microsystems, Inc. Comparado con Java, el funcionamiento de JavaScript está limitado porque no se compila antes de la ejecución. Con JavaScript es posible agregar a las páginas Web aplicaciones y funciones básicas en línea, pero el número y complejidad de las funciones de la interfaz de programación de aplicaciones disponibles es menor que con Java. El código JavaScript, que se incluye en una página Web junto con el código HTML, suele considerarse más fácil de escribir que el de Java, especialmente para los programadores menos expertos. Para interpretar el código JavaScript se requiere un explorador de Web compatible con JavaScript, como Microsoft Internet Explorer o Netscape Navigator.

JPEG

(Joint Photographic Expert Group) Formato de archivos gráficos que se utiliza para mostrar gráficos en color de alta resolución en el World Wide Web. Los gráficos JPEG aplican un esquema de compresión especificado por el usuario que reduce considerablemente los grandes tamaños de archivo asociados normalmente a los gráficos en color realistas. Cuanto mayor sea el nivel de compresión, menor será la calidad de la imagen, mientras que cuanto menor sea el nivel de compresión mayor será la calidad de la imagen.Consulte también JPEG progresivo.

JPEG progresivo

Mejora de la especificación del formato de archivos gráficos JPEG que permite mostrar gradualmente una imagen realista en un explorador de Web, mostrando versiones cada vez más detalladas de toda la imagen hasta que terminan de descargarse todos los datos de la imagen. Si bien esto es similar a los gráficos GIF entrelazados, los gráficos JPEG progresivos pueden conservar la alta calidad de color de 24 bits y ofrecen la misma eficiencia de compresión que el formato JPEG estándar. Consulte también JPEG.

LAN

(Red de área local) Tecnología de red de equipos diseñada para conectar equipos separados entre sí por una distancia corta. Es posible conectar una LAN a Internet, así como configurarla como una intranet.

Lenguaje de consulta estructurado

(SQL) Lenguaje de consulta y programación de bases de datos ampliamente utilizado para tener acceso, consultar, actualizar y administrar datos en sistemas de bases de datos relacionales. Con SQL puede recuperar datos de una base de datos, crear bases de datos y objetos de base de datos, agregar datos, modificar datos existentes y realizar otras operaciones más complejas. Mediante SQL también puede cambiar la configuración del servidor, modificar la configuración de la sesión o de la base de datos y controlar las instrucciones de datos y de acceso.

mapa de imagen

Gráfico que contiene una o más regiones invisibles, denominadas zonas activas, que tienen hipervínculos asociados. Normalmente, un mapa de imagen ofrece a los visitantes del sitio pistas visuales acerca de la información que está disponible al hacer clic en cada parte de la imagen o el gráfico. Por ejemplo, un mapa geográfico podría convertirse en un mapa de imagen si asignara zonas activas a cada región de interés del mapa.

mapa de imagen del cliente

Mapa de imagen que codifica directamente en una página Web la dirección URL de destino de cada zona activa. Los mapas de imagen del cliente no requieren procesamiento alguno por parte de un servidor para que un visitante del sitio pueda seguir los hipervínculos del mapa de imagen. Sin embargo, no todos los exploradores de Web aceptan mapas de imagen del cliente. Consulte también mapa de imagen.

mapa de imagen del servidor

Mapa de imagen que pasa las coordenadas del puntero del mouse a una rutina de controlador CGI del servidor. Los mapas de imágenes del servidor requieren que el servidor Web calcule la dirección URL de destino del hipervínculo según las coordenadas del puntero del mouse. Consulte también mapa de imagen.

marca de agua

Gráfico que aparece en los fondos de las páginas de un sitio Web para decorar e identificar las páginas, pero que no se desplaza a medida que se desplaza la página. No todos los exploradores de Web son compatibles con marcas de agua.

marcador

Ubicación con nombre en una página Web que puede ser el destino de un hipervínculo. Se puede aplicar un marcador a una cadena de caracteres o bien puede existir en una página separado de cualquier texto. Con los marcadores, los autores pueden crear vínculos a una determinada sección de una página de destino. En una dirección URL, delante del marcador se muestra un signo de número (#). También se denomina ancla.

marco

Área de una ventana del explorador de Web definida mediante una página de marcos. Un marco aparece en un explorador de Web como una de una serie de áreas diferentes en las que se pueden mostrar páginas. Un marco puede ser desplazable y dimensionable, y puede tener un borde. Para mostrar una página en un marco, cree un hipervínculo a la página y especifique el marco como parte del hipervínculo. Consulte también páginas de marco.

marco de destino

Nombre de un marco en el que se muestra la página de destino de un hipervínculo. Normalmente, un hipervínculo de un marco de una página de marcos proporcionará como su marco de destino otro marco de la página de marcos. Consulte también marco, página de marcos.

margen interior de celdas

Espacio existente entre el contenido y los bordes internos de una celda de tabla.

máscara de dirección IP

(máscara de dirección de Protocolo de Internet) Intervalo de direcciones IP definido de forma que sólo aquellos equipos cuyas direcciones IP estén dentro del intervalo puedan tener acceso a un servicio de Internet. Para enmascarar una parte de las direcciones IP, reemplácela con el carácter comodín asterisco (*). Por ejemplo, 192.44.*.* representa todos los equipos de Internet cuya dirección IP comienza con 192.44.

metaetiqueta

Etiqueta HTML que debe aparecer en la parte de la página. Las metaetiquetas proporcionan información acerca de la página, pero no afectan a su apariencia. Se utiliza una metaetiqueta estándar, "generator", para indicar el tipo de editor con el que se creó la página HTML.

Módem

Dispositivo que se conecta al ordenador y que permite intercambiar datos con otros ordenadores a través de la línea telefónica. Existen módems internos que se conectan a una ranura de expansión libre del ordenador, o módems externos que se conectan mediante un cable al puerto de serie o a un USB.

 

multi-hosting

Capacidad de un servidor Web para tener más de una dirección de Internet y más de una página principal. También se denomina multi-homing.

Navegador: 

Programa que permite visualizar el contenido de las páginas web de Internet. También se puede utilizar para descargar ficheros vía web, realizar F.T.P. o visualizar algunos elementos multimedia como sonidos, imágenes o, incluso, vídeos

news

Protocolo de Internet para recuperar archivos de un grupo de noticias de Internet.

nombre de dominio

Dirección de una ubicación de red en el formato que identifica el propietario de dicha dirección con el formato: servidor.organización.tipo. Por ejemplo, www.whitehouse.gov identifica el servidor Web de la Casa Blanca en Estados Unidos, que forma parte de la administración de ese país. Consulte también ubicación de red.

nombre de host

NTLM

(NT LanMan) Protocolo de autenticación Desafío/Respuesta de Windows NT. Este protocolo utiliza cifrado para la transmisión segura de contraseñas.

OLE

(Vinculación e incrustación de objetos) Tecnología para la transferencia y el uso compartido de información entre aplicaciones diferentes. OLE permite a un autor invocar diferentes componentes de editores para crear un documento compuesto.

P+F

(Preguntas más frecuentes) Documento que contiene una lista de preguntas y respuestas frecuentes sobre un tema concreto. En Internet se suelen exponer P+F en grupos de noticias en los que los participantes nuevos realizan las mismas preguntas que determinados lectores ya han respondido varias veces.

página

Documento de un sitio Web escrito en el lenguaje HTML.

Consulte página actual, Web actual.

Página Active Server

(ASP) Documento que contiene secuencias de comandos incrustadas del servidor. Los servidores Web compatibles con ASP pueden ejecutar estas secuencias de comandos. Para el cliente, una ASP es un documento HTML estándar que puede verse en cualquier plataforma mediante cualquier explorador de Web.

página de confirmación

Página que se muestra en el explorador de Web para confirmar que los datos introducidos en el formulario se han enviado correctamente. La página de confirmación de un formulario se especifica en el cuadro de diálogo del controlador de formulario.

página de marcos

Página que divide la ventana de un explorador de Web en regiones denominadas marcos para mostrar páginas Web de manera independiente. Consulte también marco.

página de marcos anidada

Página de marcos que contiene otra página de marcos dentro de uno de sus marcos. Consulte también página de marcos, marcos.

página inicial

Página que se muestra inicialmente en un marco cuando un visitante del sitio explora una página de marcos que contiene el marco.

página principal

En el World Wide Web, punto de entrada a un conjunto de páginas Web y otros archivos de un sitio Web. Es la página que se muestra como predeterminada cuando un visitante visita el sitio con un explorador de Web. El nombre de la página principal depende del tipo de servidor Web utilizado para alojar el sitio Web. Algunos servidores Web reservan el nombre Index.htm para la página principal, mientras que otros la llaman Default.htm.

páginas lentas

Páginas que exigen al visitante del sitio un tiempo excesivamente largo (20 segundos o más) para su descarga en un explorador de Web.

paleta independiente del explorador

Tabla de colores que presenta únicamente 216 de 256 colores posibles; se utiliza para hacer coincidir con precisión los colores de los gráficos e imágenes en exploradores de Web de diferentes plataformas. Los 40 colores restantes varían en los equipos compatibles con IBM y Macintosh y, por tanto, se omiten.

par nombre-valor

Nombre de un campo de formulario y valor del campo en el momento en que se envía el formulario. Cada campo de un formulario puede tener uno o más pares nombre-valor y el formulario propiamente dicho puede tener uno o más pares nombre-valor.

PCD

(Photo CD) Formato de archivos gráficos desarrollado por Eastman Kodak Company.

PCT

(Tecnología de comunicaciones personales) Versión mejorada de Secure Sockets Layer. Consulte también Secure Sockets Layer.

PCX

Formato de archivos gráficos que comprime los datos del gráfico con compresión de tipo RLE y que utilizaban las primeras versiones de Windows Paintbrush.

plantilla

Conjunto de formatos prediseñados para texto y gráficos en los que puede basar nuevas páginas y sitios Web. Tras crear una página o un Web con una plantilla, puede personalizar una u otro.

PNG

(Portable Network Graphics, Gráficos de red portátiles) Formato de archivo para gráficos de mapas de bits comprimidos, similar al formato GIF.

PPP

(Protocolo punto a punto) Estándar de Internet para transmitir datos a través de vínculos serie entre los equipos.

programa del cliente

En Internet, programa que se ejecuta en un equipo cliente en lugar de ejecutarse en un equipo servidor.

protocolo

Método para tener acceso a un documento o servicio a través de Internet, como el Protocolo de transferencia de archivos (FTP) o el Protocolo de transferencia de hipertexto (HTTP). También se denomina tipo.

proveedor de servicios Internet

Organización que proporciona servicios de conectividad de Internet a personas, empresas y otras organizaciones. Algunos ISP son grandes corporaciones nacionales o internacionales que ofrecen acceso desde muchas ubicaciones, mientras que otros se limitan a una ciudad o región específica.

publicar

Proceso de hacer que un Web sea público en el World Wide Web o en una intranet copiando todas sus páginas y archivos al servidor Web conectado a Internet o a la red de área local.

puerto

Uno de los canales de entrada y salida de red de un equipo que ejecuta TCP/IP. En el World Wide Web, los puertos suelen hacer referencia al número de puerto en el que se ejecuta un servidor. Un único equipo puede tener varios servidores Web en ejecución, pero en cada puerto sólo puede haber un servidor en ejecución. El puerto predeterminado de los servidores Web es 80.

RAS

Formato de archivo gráfico de barrido desarrollado por Sun Microsystems.

registro

En una base de datos, grupo de campos de información relacionados considerados como una unidad. Los registros son los equivalentes lógicos de las filas de una tabla..

relación de aspecto

En gráficos e imágenes, la relación entre el ancho o el área de una imagen con respecto a su alto. Por ejemplo, una relación de aspecto de 2:1 indica que la imagen es el doble de ancha que de alta. La relación de aspecto es un factor importante para mantener las proporciones correctas al incorporar una imagen en otro documento como una página Web.

ruta de acceso

Parte de una dirección URL que identifica las carpetas que contienen un archivo. Por ejemplo, en la dirección URL http://ejemplo.microsoft.com/hola/todos/saludos.htm, la ruta de acceso es /hola/todos/.

secuencia

En conversaciones de correo electrónico y grupos de noticias de Internet, una serie de mensajes y respuestas relacionados con un tema específico.

secuencia de comandos

Tipo de código que se puede ejecutar directamente mediante un programa que entienda el lenguaje en el que está escrita la secuencia de comandos. No es necesario compilar las secuencias de comandos para ejecutarlas.

Secure Sockets Layer

(SSL) Propuesta de estándar abierto desarrollada por Netscape Communications para establecer un canal de comunicaciones seguras y evitar la interceptación de información crítica, como números de tarjetas de crédito. La finalidad principal de SSL es permitir las transacciones financieras electrónicas seguras en el World Wide Web, aunque está diseñado para trabajar también con otros servicios de Internet. Esta tecnología, que utiliza cifrado por clave pública, está presente en muchos servidores y exploradores de Web.

selector

En una definición de estilo de hoja de estilos en cascada (o regla de estilo), el elemento HTML vinculado a un determinado conjunto de propiedades y valores de estilo.

selector de clases

En una definición de estilo de una hoja de estilos en cascada (o regla de estilo), subconjunto de selectores que controla si puede utilizarse el mismo selector para estilos ligeramente diferentes en situaciones distintas.

selector de Id.

En una definición de estilo de una hoja de estilos en cascada (o regla de estilo), selector que se utiliza para definir un estilo para un elemento individual de página, normalmente como un estilo en línea.

Servicios de Internet Information Server.

(IIS) marca de software de servidor de Microsoft que utiliza el Protocolo de transferencia de hipertexto para distribuir documentos del World Wide Web. IIS incorpora diversas funciones de seguridad, permite programas CGI y permite utilizar servidores FTP.

servidor

Equipo que ofrece servicios en una red. En el World Wide Web, el servidor es un equipo que ejecuta el software de servidor Web que responde a peticiones del protocolo HTTP. También se denomina host.

servidor de archivos

Equipo que funciona en una red y que almacena archivos y proporciona acceso a ellos. También se denomina servidor. Consulte también servidor Web.

servidor de seguridad

Método para proteger los archivos y programas de una red frente a su utilización por parte de usuarios de otra red. Un servidor de seguridad bloquea los accesos no deseados a una red protegida y ofrece a ésta acceso a otras redes externas al servidor de seguridad. Las organizaciones suelen instalar servidores de seguridad para dar acceso a los usuarios a Internet, al tiempo que protegen su información interna. 

servidor proxy

Servidor de Internet que actúa como servidor de seguridad, haciendo de mediador del tráfico entre una red protegida e Internet. Consulte también servidor de seguridad.

servidor virtual

Uno de los múltiples sitios Web que se ejecutan en el mismo servidor, cada uno de los cuales tiene un nombre de dominio único y una única dirección IP. Un servidor Web que admite servidores virtuales se denomina servidor Web multihost.

SLIP

(Protocolo Internet de línea serie) Estándar de Internet para transmitir datos a través de vínculos serie entre los equipos.

sonido de fondo

Archivo de sonido asociado con una página Web. Cuando se muestra la página en un explorador de Web, el archivo de sonido se reproduce continuamente o el número de veces especificado por la página.

subprograma Java

Clase de Java cargada y ejecutada por una aplicación Java que ya está en ejecución, como un explorador de Web. Es posible descargar y ejecutar los subprogramas Java desde un explorador de Web capaz de interpretar Java, como Microsoft Internet Explorer o Netscape Navigator. Los subprogramas Java suelen utilizarse para agregar efectos multimedia e interactividad a las páginas Web, como películas de vídeo, animaciones, calculadoras, relojes en tiempo real y juegos interactivos. Los subprogramas pueden activarse automáticamente cuando la página que los contiene se muestra en un explorador de Web o bien pueden requerir alguna acción por parte del visitante del sitio, como hacer clic en un elemento de la página.

tabla

Una o más filas de celdas en una página que se utilizan para organizar el diseño de una página o para distribuir datos sistemáticamente.

TCP

(Protocolo de control de transmisión) Software de red de Internet que controla la transmisión de paquetes de datos a través de Internet. Entre sus tareas, TCP comprueba la pérdida de paquetes, coloca los datos de varios paquetes en el orden correcto y solicita la retransmisión de los paquetes perdidos o dañados. Los equipos deben ejecutar TCP para comunicarse con los servidores Web.

TGA

(Targa) Formato de archivos gráficos realista diseñado para sistemas con un adaptador de vídeo Truevision.

TIFF

(Formato de archivo de imagen con etiquetas) Formato de archivos gráficos de alta resolución que se basa en etiquetas. TIFF se utiliza para el intercambio universal de gráficos digitales. 

tipo de archivo

Formato de un archivo, que se indica normalmente mediante su extensión de archivo. Las aplicaciones suelen trabajar con un conjunto limitado de tipos de archivo.

tipo MIME

(Tipo Extensiones multipropósito de correo Internet) Método que emplean los exploradores de Web para asociar archivos de un cierto tipo con aplicaciones que muestran archivos de dicho tipo.

Una dirección URL base debe terminar con una parte de nombre de documento, como en http://ejemplo.microsoft.com/ejemplo.htm, o con una barra diagonal, como en http://ejemplo.microsoft.com/subdir/. Consulte también dirección URL.

ubicación de red

En una dirección URL, nombre exclusivo que identifica un servidor de Internet. Una ubicación de red tiene dos o más partes, separadas por puntos, como en ejemplo.microsoft.com. También se denomina nombre de host y dirección de Internet. Consulte también dirección URL.

Los sitios Web de ensayo permiten a los autores y a los grupos de trabajo realizar cambios significativos o actualizaciones en sitios Web antes de publicarlos en el World Wide Web o en la intranet de una organización. World Wide Web

Conjunto total de documentos con hipervínculos residentes en servidores HTTP de todo el mundo. Los documentos del World Wide Web se denominan páginas o páginas Web, escritas en HTML (Lenguaje de marcado de hipertexto). Las páginas Web se identifican mediante direcciones URL (localizadores de recursos universales) que especifican el equipo concreto y la ruta de acceso en los que se puede tener acceso a un archivo y se transmiten de nodo a nodo hasta el usuario final mediante HTTP (Protocolo de transferencia de hipertexto). Las páginas Web pueden contener texto en diversas fuentes y estilos, imágenes, gráficos, clips de vídeo, sonidos y pequeños programas incrustados que se ejecutan cuando un visitante los activa haciendo clic en un hipervínculo. Los visitantes del sitio también pueden descargar archivos y enviar mensajes a otros usuarios por correo electrónico utilizando los vínculos de una página Web. El World Wide Web fue desarrollado por Timothy Berners-Lee en 1989 para el Laboratorio Europeo de física de partículas (CERN).

World Wide Web Consortium

(W3C) Consorcio de instituciones comerciales y educativas que supervisa la investigación y promueve los estándares en todas las áreas relacionadas con el World Wide Web.

zona activa

Área de un gráfico o una imagen definida gráficamente que contiene un hipervínculo. Un gráfico con zonas activas se denomina mapa de imagen. En los exploradores de Web, las zonas activas son invisibles. Los visitantes del sitio pueden saber que hay una zona activa por el cambio en la apariencia del puntero del mouse cuando éste pasa sobre el gráfico. Consulte también mapa de imagen.
